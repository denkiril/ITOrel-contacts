﻿Для работы приложения необходимо:

1. Скачать и установить OpenServer: https://ospanel.io 

2. Скопировать папку с проектом (название по умолчанию: “ITOrel-contacts”) в папку доменов OpenServer (путь по умолчанию: C:\OSPanel\domains)

3. Запустить веб-сервер (C:\OSPanel\Open Server x64.exe). В контекстном меню OpenServer в трее нажать “Запустить” (зелёный флажок).

4. Загрузить в веб-сервер прилагающуюся тестовую БД (db/contact-list.sql):
- В меню OpenServer в трее выбрать “Дополнительно” > “PhpMyAdmin”;
- Пользователь root, без пароля;
- "Создать БД" - вкладка "Импорт" - "Выберете файл" - находим в папке db проекта файл contact-list.sql - нажимаем "Вперёд".

5. Запустить веб-приложение: в меню OpenServer в трее выбрать “Мои сайты” > “ITOrel-contacts” (название по умолчанию).
